# Экспорт данных (Excel)

## Описание

Функциональность экспорта данных из админки в формате Excel (XLSX). Используется библиотека `phpoffice/phpspreadsheet`. Доступно из Filament-админки.

## Экспортёры

### OrderExcelExporter

Экспорт заказов в Excel-файл.

**Файл:** `app/Services/Export/OrderExcelExporter.php`

**Экспортируемые данные:**

- ID заказа
- Данные пользователя (ФИО, email, телефон, адрес)
- Состав заказа (товары, первая картинка товара, количество, цены)
- Итоговые суммы (без скидки, со скидкой)
- Статус заказа
- Промокод (если применён)
- Дата создания

**Особенности изображений в экспорте заказа:**

- В таблицу добавляется отдельная колонка `Фото`.
- Берётся первое изображение товара из media-коллекции `images`.
- Для уменьшения веса файла используется уменьшенная версия изображения (`thumb`) с fallback на оригинал.
- Если у товара нет изображения, в колонке отображается `—`.

### StockExcelExporter

Экспорт данных о складских остатках.

**Файл:** `app/Services/Export/StockExcelExporter.php`

**Экспортируемые данные:**

- ID товара
- Название товара
- Slug
- Цена
- Количество на складе (stock)
- Статус наличия

## Использование

Экспорт доступен из Filament-админки через кнопки действий (Actions) в соответствующих ресурсах (Orders, Products).

## Зависимости

- `phpoffice/phpspreadsheet ^5.5` (composer.json)
