# Спецификации проекта "Магазин магических товаров"

Этот каталог содержит спецификации (specs) для различных функций проекта. Каждая спецификация включает:

- `requirements.md` - требования и пользовательские истории
- `design.md` - архитектурное решение и дизайн
- `tasks.md` - план реализации с задачами

## Существующие спецификации

### Основные функции
- `product-catalog` - Каталог товаров с фильтрацией и поиском
- `user-authentication` - Система аутентификации пользователей
- `order-management` - Управление заказами
- `category-management` - Управление категориями товаров

### Дополнительные функции
- `admin-panel` - Административная панель на Filament
- `api-documentation` - Автоматическая генерация документации API

## Структура проекта

Проект построен на Laravel 11 с использованием:
- PHP 8.3+
- Filament 3 для админ-панели
- Sanctum для API аутентификации
- Scribe для документации API
- Docker для разработки
- Pest для тестирования

## Архитектурные принципы

1. **Repository Pattern** - для работы с данными
2. **Service Layer** - для бизнес-логики
3. **API Resources** - для трансформации данных
4. **Form Requests** - для валидации
5. **Строгая типизация** - все классы используют `declare(strict_types=1)`
